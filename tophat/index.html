Tophat

Tophat is a Clojure library for handling HTTP request and response documents/maps.

Source Repository

The tophat git repository is here.

API Documentation

The Clojure API documentation is here.

Rationale

Tophat began as a utility library that was passed between projects to handle HTTP request and response documents. As the library evolved, the HTTP response maps (aka Ring maps) were being used more and more as results from function calls to handle exception handling so it made sense to break out Tophat into a top-level library/project.

Usage

In project.clj dependencies: 

[tophat "0.1.2"]

In code: 

(ns your-ns
  (:require [tophat.core :as tophat]))
