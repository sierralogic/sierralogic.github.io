Diction

Diction is a Clojure library for defining schema/specs, validation, generative testing, grooming, generated sensible human-readable values, custom data elements, guarded functions, generative test function generation, HTTP payload/parameter validation, decoration/undecoration, and navigatable data diction generation of HTML, markdown, and hiccup.
